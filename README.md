# Best Shows

A modern, minimal website for TV and movie reviews. Built with pure HTML & CSS.

## Design Features

- Dark theme with blue accent
- Responsive layout (mobile + desktop)
- Smooth animations and hover effects
- Clean typography with Inter font
- Categories, tags, and rating system
- "Essential Tier" top picks section

## Quick Start

1. Edit `index.html` to customize content
   - Replace review text, titles, ratings
   - Update about section and contact email
2. Deploy to GitHub Pages:
   - Push to GitHub
   - Enable Pages in repo settings (source: main branch / root)
   - Site will be live at `https://yourusername.github.io/bestshows/`

## Customize

- Colors: edit `:root` variables in `style.css`
- Add/remove reviews: duplicate `<article class="review-card">` blocks
- Change rating thresholds or add categories as needed
